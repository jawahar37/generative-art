Generative art

A gallery of art generated with code and rendered using HTML5 Canvas.
This project is inspired by the Canvas Cards collection by Luke Patton.

10 PRINT

10 PRINT is a program written in the early 80s to run on a COMMODORE 64 system to generate a beautiful maze with a simple condition.10 PRINT CHR$ (205.5 + RND (1)); : GOTO 10It prints either a \ or a / with a 50% probability for each.

Square bloom

Draw the biggest square you can at a random place that doesn't touch any existing squares. Repeat till it becomes hard to find a new spot.
